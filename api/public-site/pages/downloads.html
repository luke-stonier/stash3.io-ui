Need help installing?

Windows
  1. Download the .exe or .msi.
  2. Run the installer and follow prompts.
  3. Launch Stash3 from Start menu.
macOS
  1. Open the .dmg and drag to Applications.
  2. If Gatekeeper warns, open via System Settings → Privacy & Security.
  3. Launch Stash3 from Applications.
Linux
  1. Make AppImage executable: chmod +x Stash3.AppImage
  2. Run with ./Stash3.AppImage or install the .deb/.rpm.
  3. Launch from your app menu.
